Mismatch Notes

AI Assistant / Learn Studio

Color says Purple for Intelligence / AI.

  • AI Assistant currently sits in Technology / System.
  • Learn Studio currently sits in Media / Audio / Community.

Those two probably belong together.

Cloud Studio / Storage Inspector / Publish Studio

Color says Blue for Platform / Cloud / Storage.

  • Cloud Studio currently sits in Technology / System.
  • Storage Inspector currently sits in Technology / System.
  • Publish Studio currently sits in Settings and Admin.

These three naturally form a Platform group.

Marketplace

Color says Gold for Community / Distribution.

Marketplace currently sits in Media / Audio / Community, which fits well.

Palette Manager

Color says Orange for Utility / Configuration.

Palette Manager currently sits in Assets / Content, which is the better placement because that is how users will think about it.

Personally, the current category names are better, but the badge colors should be used as a secondary visual grouping inside those categories, not as the primary grouping mechanism. That gives users meaningful categories while still getting instant color recognition.

Recommended future alignment: keep the current category names as the primary user-facing grouping. Badge colors should be secondary visual grouping cues inside those categories, and tile colors can be corrected later.
Shared grouping color layer: grouping colors are applied through shared group classes and per-group imports under assets/css/tools/grouping/. Tools can be rethemed by changing the grouping CSS import or token mapping, and pages should not define their own group colors. The current mapping follows the Color-Aligned Grouping Option: Build & Create pink, Content & Assets orange, Media & Audio red, AI & Learning purple, Platform & Cloud blue, and Development & System gray.
