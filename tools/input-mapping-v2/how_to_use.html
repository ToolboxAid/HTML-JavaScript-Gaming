Input Mapping V2

Select an action, capture a keyboard key, combo, mouse button, pointer drag, or gamepad input, then review the generated mapping JSON.

  1. Open Input Mapping V2.
  2. Select an action from the Actions panel.
  3. Use Capture Keyboard, Capture Combo, Capture Mouse, pointer drag, or Capture Gamepad.
  4. Click a captured input token to delete that binding.
  5. Use Copy JSON or Export when the mapping is ready.
