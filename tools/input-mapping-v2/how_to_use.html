Input Mapping V2

Select an action, enable devices, choose a gesture pattern, then capture keyboard, mouse, wheel, drag, or game controller input.

  1. Open Input Mapping V2.
  2. Select an action from the Actions panel.
  3. Use Devices and Gestures to choose supported input patterns.
  4. Use Capture Keyboard, Capture Mouse, or a connected game controller capture button.
  5. Click a captured input token to delete that binding.
  6. Use Copy JSON or JSON when the mapping is ready.
