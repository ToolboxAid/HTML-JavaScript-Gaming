How to use

Input Mapping V2

Create keyboard and gamepad bindings for games, tools, and samples.

Open Tool Read Me Tools

Workflow

  1. Select an action from the Actions panel.
  2. Use Capture Keyboard, then press the key to map.
  3. Use Capture Gamepad after pressing or holding the gamepad input to map.
  4. Review the generated mapping JSON and copy it when needed.
