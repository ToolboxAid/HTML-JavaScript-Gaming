World Vector Studio V2

Place assets, maps, layers, parallax, and scene/world layout. Build the place.

Use This For

Use World Vector Studio V2 for placing assets, maps, layers, parallax, and scene/world layout across terrain, tile/world geometry, environment composition, collision/world regions, level layout, spawn placement, and camera zones.

Architectural Split

World Vector Studio V2: Build the place. It owns the map, world, and environment structures where game objects appear and move.

Object Vector Studio V2: Build the thing. It owns reusable gameplay entities, ships, enemies, pickups, actors, weapons, UI vector assets, SVG/vector authoring, primitives/shapes/path editing, and animation-ready structures.

Object Vector Studio V2 asset references are read-only in World Vector Studio V2. World Vector Studio V2 must not mutate Object Vector Studio V2 source assets; use Duplicate As Local in Object Vector Studio V2 before editing detached object data.

Migration Direction

World Vector Studio V2 replaces Vector Map Editor for current map, layer, parallax, and world layout work. Object Vector Studio V2 replaces SVG Asset Studio for current vector asset authoring.

Examples

Platformer level/parallax scene -> World Vector Studio V2.

Spawn/camera/collision zones: World Vector Studio V2.

Asteroids ship/UFO -> Object Vector Studio V2.

Current State

This tool is currently a copied First-Class Tool V2 starter shell with World Vector Studio V2 naming and documentation. Runtime world editing behavior is planned but not implemented in this documentation PR.
