Skin Editor: How To Use

Edit game skin JSON for Breakout, Pong, Solar System, and Bouncing Ball.

Details

Skin Editor manages color/size skin payloads for arcade games and stores overrides in browser local storage for immediate play-test loops.

Tool vs Workspace Saves

Tool save/export
Action: Export skin JSON files from Skin Editor when you want disk artifacts.
Why this step matters: Creates sharable skin files you can version or move between environments.

Workspace save
Action: Save workspace state separately in Workspace Manager.
Why this step matters: Workspace save captures orchestration context, while skin overrides remain tool/browser state unless exported.

Workflow

Tool Mode

1. Select a game
Action: Choose Breakout, Pong, Solar System, or Bouncing Ball in the Game dropdown.
Why this step matters: Targets the correct skin schema and runtime destination.

2. Load active skin
Action: Click Load Active Skin.
Why this step matters: Pulls the current skin from local override (if present) or the default game skin file.

3. Edit and apply override
Action: Update JSON and click Apply Skin Override.
Why this step matters: Stores the override so the game uses your edited skin immediately.

4. Verify in-game
Action: Click Open Game and validate visuals.
Why this step matters: Confirms edits in live gameplay rendering.

Workspace Mode

1. Launch from Workspace Manager
Action: Open Skin Editor in workspace context for your active game flow.
Why this step matters: Keeps tool usage aligned with your broader workspace session.

2. Apply skin override
Action: Update skin JSON and apply override for the selected game.
Why this step matters: Lets you test visual updates while staying in the workspace loop.

3. Open game and validate
Action: Launch the game and confirm skin changes.
Why this step matters: Ensures game output matches edited payload.

4. Export when ready
Action: Export final skin JSON for sharing/versioning.
Why this step matters: Preserves the approved skin artifact outside browser storage.

Buttons

Apply Skin Override #skinEditorSaveOverrideButton
Details: Saves the current JSON as the active local override for the selected game.

Clear Override #skinEditorClearOverrideButton
Details: Removes local override and reloads the default skin source.

Export Skin JSON #skinEditorExportButton
Details: Downloads the current skin document.

How to Use #skinEditorHowToUseButton
Details: Opens this page.

Import Skin JSON #skinEditorImportInput
Details: Loads a local JSON file into the editor for review/apply.

Load Active Skin #skinEditorLoadButton
Details: Loads active skin from override or game file.

Open Game #skinEditorOpenGameButton
Details: Launches the selected game for live visual verification.
