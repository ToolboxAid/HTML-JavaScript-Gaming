    Layers

      Collision and Data layers are first-class map layers in this foundation. Parallax remains reserved for a separate companion editor.

      Usage

      • Paint and Erase write to the selected layer.
      • Picker reads current cell value into active tile.
      • Marker tool places object or spawn markers per cell.
      • Right click on the canvas erases or removes a marker.
      • Use Simulate to preview tile/data/object state in motion.
      • Saved JSON keeps a reserved parallax block for future companion tooling.
