Palette Manager V2: How To Use

User Palette

Add or select user swatches, then use Add to edit symbol, hex, and name before updating the selected swatch. User-created swatches use User Added as their source.

Selected Swatch

Selected Swatch displays symbol, hex, name, source, and tags. Click a selected tag to remove it from the selected swatch.

Add

Add shows symbol, hex, and name form data only when the selected user palette swatch was created as User Added. Source-palette selections clear the form data, and duplicate names, RGB/hex values, or symbols are blocked.

Tags

Tags lists existing user palette tags. Click a tag to toggle it on the selected swatch, or enter a new tag name and add it to the selected swatch. Tags are stored in lowercase.

Browse Palettes

Choose Crayola, W3C, or JavaScript, search the browse-only list, and use the red tack to copy a swatch into the user palette.

Pinning

A red tack copies a source swatch into tools.palette-manager-v2.swatches with its source. A green tack attempts removal and blocks if isSwatchUsedByTool(swatch) reports usage.

Import/Export

Import must be valid JSON with tools.palette-manager-v2.swatches. Export and copy emit that same global key, including optional tag arrays when present.
