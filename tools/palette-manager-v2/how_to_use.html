Palette Manager V2: How To Use

User Palette

Add or select user swatches, then use User Defined Swatch to edit symbol, hex, name, and tags before updating or removing the selected swatch. User-created swatches use User Added as their source.

Selected Swatch

Selected Swatch displays symbol, hex, name, and source as read-only details. Tags remain editable there and offer suggestions from tags already used in the active palette.

Browse Palettes

Choose Crayola, W3C, or JavaScript, search the browse-only list, and use the red tack to copy a swatch into the user palette.

Pinning

A red tack copies a source swatch into tools.palette-browser.swatches with its source. A green tack attempts removal and blocks if isSwatchUsedByTool(swatch) reports usage.

Import/Export

Import must be valid JSON with tools.palette-browser.swatches. Export and copy emit that same global key, including optional tag arrays when present.
