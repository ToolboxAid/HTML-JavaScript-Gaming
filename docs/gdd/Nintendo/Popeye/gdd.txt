# 1. 🧭 CORE GAME OVERVIEW

* **Item Collection Under Pressure**
* **Dynamic Hazard Avoidance**
* **Enemy Pursuit (Bluto)**
* **Multi-Level Navigation**
* **Short, High-Intensity Rounds**

---

## 1.2 Player Fantasy

You are Popeye collecting items thrown by Olive Oyl while avoiding Bluto and environmental hazards across a multi-platform arena.

---

# 2. 🎮 CORE GAMEPLAY LOOP

1. Spawn in arena
2. Olive throws collectible items
3. Navigate platforms and ladders
4. Collect items before they fall off
5. Avoid Bluto and hazards
6. Clear all items → advance stage
7. Difficulty increases

---

# 3. 🧍 PLAYER SYSTEM (POPEYE)

## 3.1 Controls

* Left / Right movement
* Climb up/down ladders
* Punch (limited use, contextual)

---

## 3.2 Movement Model

```text
velocityX = input * speed
velocityY = ladderSpeed (when climbing)
gravity applies when falling
```

---

## 3.3 Movement Properties

```text
Horizontal Speed: Moderate
Climb Speed: Fixed
Jump: None (CRITICAL DIFFERENCE from Donkey Kong)
```

---

## 3.4 Player States

| State    | Description              |
| -------- | ------------------------ |
| Walking  | Ground/platform movement |
| Climbing | Ladder movement          |
| Falling  | Drop between levels      |
| Punching | Attack                   |
| Dead     | Life lost                |
