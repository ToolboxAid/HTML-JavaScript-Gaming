* **Timing-Based Dodging & Punching**
* **Telegraphed Attacks**
* **Stamina & Knockdown System**
* **Memorization + Reaction Skill**

---

## 1.2 Player Fantasy

You are a rising boxer (Little Mac style), defeating larger opponents by reading patterns and striking at the right moment.

---

# 2. 🎮 CORE GAMEPLAY LOOP

1. Match begins
2. Opponent enters idle pattern
3. Opponent telegraphs attack
4. Player dodges or blocks
5. Player counters with punches
6. Build damage / earn stars
7. Knock opponent down
8. Win match → advance

---

# 3. 🧍 PLAYER SYSTEM

## 3.1 Controls

* Left Punch
* Right Punch
* Dodge Left
* Dodge Right
* Duck (optional depending version)
* Star Punch (special)

---

## 3.2 Movement Model

* Player stays in fixed position
* Only lateral dodging and ducking

---

## 3.3 Player States

| State        | Description      |
| ------------ | ---------------- |
| Idle         | Ready            |
| Punching     | Attack animation |
| Dodging      | Avoiding attack  |
| Hit          | Stunned          |
| Knocked Down | On canvas        |
| KO           | Match loss       |

---

# 4. 🥊 COMBAT SYSTEM

## 4.1 Punch Types

| Type       | Description      |
| ---------- | ---------------- |
| Left Jab   | Fast, low damage |
| Right Jab  | Fast, low damage |
| Hook       | Medium damage    |
| Star Punch | High damage      |

---

## 4.2 Punch Logic

```text
IF opponent vulnerable:
    apply damage
ELSE:
    block or countered
```

---

## 4.3 Star System (CRITICAL)

```text
successful timing → earn star
use star → powerful punch
```

---

## 4.4 Damage System

* Each hit reduces opponent stamina
* Vulnerability windows critical
