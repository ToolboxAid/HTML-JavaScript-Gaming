4. Avoid traffic and obstacles
5. Overtake opponents
6. Finish race → advance to next track
7. Fail checkpoint → game over

# 3. 🚗 PLAYER SYSTEM

## 3.1 Controls

* Steering (left/right)
* Acceleration
* Brake
* Gear shift (optional manual/automatic)

---

## 3.2 Movement Model (ARCADE PHYSICS)

```text id="3jwq8y"
velocity += acceleration
velocity -= friction
velocity -= brakeForce (if braking)
```

---

## 3.3 Handling Properties

```text id="9l3p2e"
Top Speed: Very high
Acceleration: Fast ramp
Turning: Loose / drift-like
Grip: Low (arcade sliding)
```

---

## 3.4 Steering Model

```text id="p7k2x1"
turnRate = baseTurn * (1 - speedFactor)
```

* Higher speed → reduced turning precision

---

## 3.5 Drift Behavior

* Slight slide during turns
* Oversteer at high speed
* No realistic traction model

---

## 3.6 Player States

| State     | Description     |
| --------- | --------------- |
| Driving   | Normal          |
| Drifting  | Turning hard    |
| Colliding | Hit object      |
| Spinning  | Loss of control |
| Finished  | Race complete   |
