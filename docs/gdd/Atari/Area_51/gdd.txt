# 🎮 `gdd.txt` — AREA 51 (Reverse Engineered GDD)

---

## 1. GAME OVERVIEW

**Title:** Area 51
**Genre:** On-Rails Light Gun Shooter
**Platform:** Arcade (Original), later ports (PS1, PC)
**Developer:** Atari Games
**Release Year:** 1995

### Core Concept

A military containment team is deployed into a top-secret facility overrun by alien entities and infected personnel. Players progress through predefined paths, eliminating threats, rescuing survivors, and uncovering escalating extraterrestrial horrors.

---

## 2. CORE GAMEPLAY LOOP

1. Player enters scene (fixed camera)
2. Enemies emerge from scripted spawn points
3. Player:

   * Shoots enemies
   * Avoids civilians
   * Searches for hidden items
4. Scene timer counts down
5. Completion conditions:

   * Kill quota reached OR
   * Timer expires
6. Transition to next scene

### Failure Conditions

* Player health reaches zero
* Critical hostage deaths (optional design expansion)
* Continue required

---

## 3. INPUT & CONTROLS

| Action | Input              |
| ------ | ------------------ |
| Aim    | Light gun tracking |
| Shoot  | Trigger            |
| Reload | Shoot off-screen   |
| Start  | Button             |

### Expanded (Home Ports)

* Controller aim via crosshair
* Manual reload button

---

## 4. PLAYER SYSTEMS

### Health System

* Starts at 100 HP
* Damage per hit varies (10–25)
* No regeneration
* Health packs: +25 / +50

### Weapons (Reverse Engineered + Expanded)

| Weapon         | Type    | Notes          |
| -------------- | ------- | -------------- |
| Standard Rifle | Default | Unlimited ammo |
| Shotgun        | Pickup  | Spread damage  |
| Machine Gun    | Pickup  | High fire rate |
| Alien Weapon   | Rare    | High damage    |

### Shooting Model

* Hitscan (no projectile travel)
* Per-frame collision check
* Weak points:

  * Head = instant kill
  * Body = standard damage

---

## 5. ENEMY DESIGN

### Enemy Categories

#### 5.1 Infected Personnel

* Fast, humanoid
* Attack: melee
* Behavior: rush player

#### 5.2 Alien Grunts

* Medium speed
* Ranged attacks
* Take multiple hits

#### 5.3 Facehuggers

* Small, fast targets
* Jump toward camera
* High difficulty to hit

#### 5.4 Turrets / Machines

* Stationary
* Predictable firing pattern

#### 5.5 Boss Creatures

* Large hitboxes
* Multi-phase
* Weak points required

---

## 6. AI BEHAVIOR SYSTEM

### State Machine

* Idle (hidden)
* Spawn animation
* Attack phase
* Hit reaction
* Death animation

### Targeting Logic

* Always aimed at player camera
* Predictive attack timing (illusion of intelligence)

---

## 7. SCORING SYSTEM

| Action            | Points     |
| ----------------- | ---------- |
| Enemy kill        | 100–500    |
| Headshot          | Bonus +200 |
| Civilian hit      | -500       |
| Hidden item found | +1000      |

### Combo System (Expanded)

* Consecutive hits without miss = multiplier
* Encourages accuracy
