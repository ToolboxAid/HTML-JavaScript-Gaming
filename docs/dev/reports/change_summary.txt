Summary
- Added a new standalone SVG Background Editor at `tools/SVG Background Editor/` for background art authoring.
- Implemented SVG authoring foundation with:
  - New/Load/Save SVG workflow
  - local sample loading from `tools/SVG Background Editor/samples/`
  - shape tools: rectangle, ellipse/circle, line, polyline, and path
  - fill/stroke/stroke-width controls
  - element selection, move, resize, delete
  - element list with order controls (send backward / bring forward)
  - canvas zoom and pan controls
- Added local SVG learning samples and a sample manifest:
  - sky gradient scene
  - mountain range scene
  - Mario-style learning backdrop
- Added a Tools hub card entry so the editor is discoverable from `tools/index.html`.

Boundary and compatibility notes
- Tool remains focused on background art authoring only.
