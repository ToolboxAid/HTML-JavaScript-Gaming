PR 11.120 - Tools Index Relationship Count

Files searched
- tools/index.html
- tools/renderToolsIndex.js
- samples/metadata/samples.index.metadata.json

Before
- tools/renderToolsIndex.js counted sample?.toolsUsed when present, otherwise sample?.toolHints.
- It also excluded the Workspace Manager card from Samples (x), which produced the user-observed 30 while samples/index.html showed 31.
- BUILD-reported tools/index relationship count: 34.

After
- tools/renderToolsIndex.js reads samples/metadata/samples.index.metadata.json.
- It counts only relationships where:
  - sample.phase is not 20
  - roundtripToolPresets has toolId
  - roundtripToolPresets has presetPath
  - toolHints contains the same toolId
  - tool id is deduplicated per sample
- Workspace Manager is counted when the metadata SSoT explicitly contains a workspace-manager preset.
- Sample 1902 now contributes exactly one count to workspace-manager.
