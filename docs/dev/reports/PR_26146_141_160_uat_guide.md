# MIDI Studio V2 UAT Guide

1. Open MIDI Studio V2 in tool-only mode.
2. Use Import JSON Manifest to load a valid game manifest or MIDI Studio toolState JSON.
3. Select a song in Song Setup.
4. Edit Name, Classification, Tempo, Key, Style, and Notes.
5. Confirm Generated ID updates as `camelCase(Name) + "-" + Classification`.
6. Populate Intro, Verse, Chorus, Bridge, Outro, or custom sections.
7. Confirm Available Sections only shows populated sections.
8. Build Song Sequence with Add, Duplicate, Move Up, Move Down, and Remove.
9. Use Parse Guided Song Sheet or Regenerate Arrangement.
10. Confirm Octave Timeline section labels, colors, notes, diagnostics, and JSON Details update.
11. Edit notes on the Octave Timeline canvas with hover, click, drag-paint, and drag-erase.
12. Select instruments from Octave Timeline and Instruments.
13. Edit instrument settings from Instruments only.
14. Audition the instrument keyboard.
15. Use Play, Loop, Stop, Play Section, and Play Sequence.
16. Assign Game Usage where needed.
17. Use Export JSON for canonical toolState export.
18. Review Export readiness and Manifest readiness.
19. Treat red/unwired controls as future work unless they become fully wired in a later PR.
