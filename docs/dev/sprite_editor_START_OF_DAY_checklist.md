# Start of Day Checklist

## Before any new work
- [ ] Open the editor
- [ ] Confirm there are no console errors
- [ ] Confirm the canvas is visible and centered
- [ ] Confirm timeline is visible
- [ ] Confirm playback controls are visible
- [ ] Confirm File / Edit / Frame / Layer menus open with visible items
- [ ] Confirm ESC closes UI surfaces in the right order
- [ ] Confirm command palette opens and closes correctly

## Quick function smoke test
- [ ] Draw with brush
- [ ] Undo
- [ ] Redo
- [ ] Select a frame
- [ ] Play animation
- [ ] Toggle layer visibility
- [ ] Open File menu and confirm actions give feedback

## Before asking Codex to code
- [ ] ChatGPT created the plan / PR text first
- [ ] Scope is small and explicit
- [ ] Requested fix does not rewrite architecture
- [ ] Requested fix is visible and testable
- [ ] Requested fix does not add clutter unless explicitly intended

## Stop rule
- [ ] If any item above fails, stop feature work and fix visibility / interaction / consistency first
