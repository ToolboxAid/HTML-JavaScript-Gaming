# Start Of Day Copy Map

Use this folder as a quick copy/paste guide for session startup.

## Session Mode
Default mode is:
- PLAN-heavy
- BUILD-light
- APPLY-fast

Use this folder to reduce unnecessary Codex runs and keep BUILD scope surgical.

## Copy To Tool (If Present In This Folder)
- `CODEX_SESSION_PROMPT.md` -> paste into the Codex command/session prompt input
- `SESSION_CONTEXT.md` -> paste into session context/background notes
- `CURRENT_STATE.md` -> paste into "current status" or kickoff summary
- `NEXT_STEPS.md` -> paste into planning/next-command drafting

If one of these files is missing, skip it and use the active files in `docs/dev/` instead.

## Active Dev Control Files
- `../codex_commands.md` -> paste into the active "COMMAND" request
- `../commit_comment.txt` -> paste as the commit comment/message text
- `../next_command.txt` -> paste as the suggested next command

## Reporting Files
- `../reports/change_summary.txt` -> copy into change-summary output section
- `../reports/validation_checklist.txt` -> copy into validation output section
- `../reports/file_tree.txt` -> copy into file-tree output section
