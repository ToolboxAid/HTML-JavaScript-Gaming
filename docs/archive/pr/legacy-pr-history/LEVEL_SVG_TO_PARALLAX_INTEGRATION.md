# LEVEL_SVG_TO_PARALLAX_INTEGRATION

## Relationship
The SVG Background Editor does not replace the Parallax Editor.

## Intended Flow
1. Draw or edit background art in SVG Background Editor
2. Save SVG asset
3. Load that SVG asset into the Parallax Editor as a layer image/source
4. Tune depth, scroll factor, repeat, and foreground/background placement in Parallax Editor

## Rule
SVG Background Editor owns art creation.
Parallax Editor owns visual depth placement and motion behavior.
