# LEVEL_SAMPLE_NUMBERING_AND_FOLDER_RULES

## Numbering Format
Use four digits:
- digits 1-2 = level
- digits 3-4 = sample number within the level

Examples:
- `0101`
- `0102`
- `0221`
- `1103`

## Folder Format
Group samples under phase folders, for example:
- `Phase 01 - Core Engine (0101-0124)/`

## Intent
This keeps:
- the root sample list short
- same-level samples together
- future expansion manageable
