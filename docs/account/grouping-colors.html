Future Alignment Note

Current group names remain the primary user-facing grouping. Badge colors are secondary visual cues. Tile colors can be corrected later without changing the grouping model.

Shared Grouping Color Layer

Grouping colors are applied by shared grouping classes and per-group imports under assets/css/tools/grouping/. Tools can be rethemed by changing the grouping CSS import or token mapping. Pages should not define their own group colors. The current mapping follows the Color-Aligned Grouping Option: Build & Create pink, Content & Assets orange, Media & Audio red, AI & Learning purple, Platform & Cloud blue, and Development & System gray.
