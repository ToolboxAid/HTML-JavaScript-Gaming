GameFoundryStudio Design System

Source Of Truth

  • GameFoundryStudio/assets/css/colors.css owns all approved color values.
  • GameFoundryStudio/assets/css/tokens.css owns non-color design tokens.
  • GameFoundryStudio/assets/css/base.css stays limited to reset, typography defaults, focus, accessibility, and base element rules.
  • GameFoundryStudio/assets/css/gamefoundrystudio.css owns shared component and pattern visuals.
  • GameFoundryStudio/assets/css/pages.css should remain page layout oriented and use shared tokens.

Token Families

  • Spacing: xs, sm, md, lg, xl, 2xl.
  • Sizing: xs, sm, md, lg, xl, 2xl.
  • Typography: base/code font families, size scale, weights, and line heights.
  • Radius: xs, sm, md, lg, xl, 2xl, and pill.
  • Shadows, z-index, icon sizes, and tool tile dimensions live in token variables.

Color Safety

New visual work must use existing color tokens. If a color is not approved, add the smallest necessary token to colors.css and document why it exists.
