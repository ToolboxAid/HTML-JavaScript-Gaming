# LEVEL_SVG_EDITOR_PALETTE_DROPDOWN

## Requirement
Add a palette dropdown to the SVG Background Editor.

## Behavior
- dropdown selects the active palette set
- selected palette applies to both Paint and Stroke selection workflow
- used-color swatches remain above the palette area
- Active Paint and Active Stroke remain stored and unchanged until intentionally changed
- palette dropdown should be available before shape styling actions are enabled

## Minimum UI Labels
- Palette
- Active Paint
- Active Stroke
- Used Colors
