# LEVEL_ALL_TOOLS_THEME_RULE

## Rule
All tools must use the shared theme system.

## Requirement
Tool UIs should not ship with isolated hard-coded visual styles when the shared theme system already exists.

## Expected Behavior
- shared colors and tokens
- shared panel styling
- shared button/input treatment
- shared selected/hover/disabled states
- tool-specific styling only where functionally necessary

## Tools Covered
- Tile Map Editor
- Parallax Editor
- SVG Background Editor
- Sprite Editor
