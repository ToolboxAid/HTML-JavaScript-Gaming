# LEVEL_EDITOR_ADVANCED_SIMULATION_ACCEPTANCE

## Acceptance Criteria
- both editors can simulate across a full map
- both editors can show meaningful scroll/motion context
- Tile Map Editor can visibly validate obstacles and authored layout over distance
- Parallax Editor can visibly validate depth and foreground/background behavior over distance
- simulation has play/pause/exit controls
- simulation does not silently alter authored project data
- no engine core API modifications

## Non-Goals
- full playable game inside editors
- combat, AI, or full gameplay systems
- merging Tile Map Editor and Parallax Editor into one tool
