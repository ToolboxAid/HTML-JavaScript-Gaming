# LEVEL_EDITOR_ACCEPTANCE_FIXES

## Acceptance Criteria
- SVG bounded boxes align to the actual drawn object with no offset
- SVG editor includes a palette dropdown
- SVG paint/stroke still use stored active color state

- Tile Map Editor supports multiple PNG tile imports and tileset-sheet imports
- imported PNGs are selectable as tile entries
- no engine core API changes
