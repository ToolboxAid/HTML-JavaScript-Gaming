# 3. 🧍 PLAYER SYSTEM

## 3.1 Movement System (CORE MECHANIC)

### No Jump Rule

* Player **cannot jump**
* Movement relies entirely on:

  * Walking
  * Falling
  * Grappling

---

## 3.2 Bionic Arm System

### Arm Directions

```text
Up
Up-Left
Up-Right
Left
Right
```

### Arm States

* Extending
* Latched
* Swinging
* Retracting

---

## 3.3 Grapple Physics

### Key Rules

* Arm attaches only to valid surfaces
* Player swings in arc (pendulum physics)
* Release timing determines trajectory

### Swing Behavior

```text
Anchor point = fixed
Player rotates around anchor
Velocity influenced by:
  - gravity
  - prior momentum
```

---

## 3.4 Movement Properties

```text
Walk Speed: Moderate
Gravity: Strong
Swing Momentum: High influence
Fall Speed: Fast
```

---

## 3.5 Combat System

### Weapon

* Default rifle (horizontal fire)
* Can shoot while:

  * Standing
  * Hanging
  * Swinging (limited)

### Fire Rules

* Single direction (forward)
* No vertical shooting (key limitation)

---

## 3.6 Player States

| State     | Description     |
| --------- | --------------- |
| Idle      | Standing        |
| Walking   | Ground movement |
| Falling   | No grapple      |
| Grappling | Arm extended    |
| Swinging  | Attached        |
| Climbing  | Vertical pull   |
| Shooting  | Firing weapon   |
| Hit       | Knockback       |
| Dead      | Life lost       |

# 4. 🧲 GRAPPLE SYSTEM (CRITICAL)

## 4.1 Valid Attach Points

* Ceilings
* Platforms (underside)
* Specific surfaces only

## 4.2 Invalid Attach Points

* Air
* Decorative backgrounds
* Some enemy surfaces

---

## 4.3 Grapple Timing

```text
Press → extend
Hit surface → latch
Hold → swing
Release → launch
```

---

## 4.4 Constraints

* Fixed arm length
* No mid-air direction change without re-grapple
* Swing arc must be mastered
