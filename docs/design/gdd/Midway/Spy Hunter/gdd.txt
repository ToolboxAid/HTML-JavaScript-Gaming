# 3. 🚗 PLAYER SYSTEM

## 3.1 Vehicle: Interceptor

### Controls

* Left / Right steering
* Acceleration (implicit via scroll)
* Weapons fire

---

## 3.2 Movement Model (CRITICAL)

```text id="3u1b6n"
velocityY = constant (forward scroll)
velocityX += steeringInput * turnRate
velocityX -= friction
```

---

### Properties

```text id="k3c8q5"
Forward Speed: Constant increasing
Lateral Speed: Limited
Acceleration: Minimal (mostly constant speed)
Drift: Slight inertia
```

---

## 3.3 Handling Feel

* Tight but slightly slippery
* Oversteering possible at high speed
* Road edges punish mistakes

---

## 3.4 Player States

| State     | Description        |
| --------- | ------------------ |
| Driving   | Normal             |
| Shooting  | Weapons active     |
| Exploding | Death              |
| Docking   | With weapons truck |
