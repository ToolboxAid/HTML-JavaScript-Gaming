# 1. 🧭 CORE GAME OVERVIEW

## 1.1 Game Pillars

* **Discrete Grid Movement**
* **Strict Timing Windows**
* **Lane-Based Hazard Systems**
* **Pattern Recognition**
* **Escalating Difficulty**

---

## 1.2 Player Fantasy

You are a frog trying to cross a dangerous road and river to reach safety.

---

# 2. 🎮 CORE GAMEPLAY LOOP

1. Player spawns at bottom
2. Navigate road (avoid cars)
3. Navigate river (use logs/turtles)
4. Reach goal slot
5. Repeat until all slots filled
6. Advance level (increase speed/difficulty)
7. Lose all lives → game over

---

# 3. 🐸 PLAYER SYSTEM

## 3.1 Movement System (GRID-BASED)

### Controls

* Up
* Down
* Left
* Right

### Movement Rules

* Moves **one tile per input**
* Instant snap to grid
* No partial movement

---

## 3.2 Movement Properties

```text
Tile Size: Fixed (e.g., 16x16 or 32x32)
Move Speed: Instant tile transition
Input Buffer: None (arcade strict)
```

---

## 3.3 Player States

| State        | Description       |
| ------------ | ----------------- |
| Idle         | Waiting for input |
| Moving       | Transition tile   |
| Riding       | On log/turtle     |
| Drowning     | Water death       |
| Crushed      | Car collision     |
| Reached Goal | Success           |
| Dead         | Life lost         |

---

## 4.1 Lane Types

### Road Section

* Cars / trucks

### River Section

* Logs

### Safe Zones

* Start area
* Median strip
* Goal row

---

## 4.2 Lane Behavior

```text
Each lane:

- Has speed
- Has entity type
- Loops infinitely
```

## 5.2 Collision Rules

# 6. 🌊 RIVER SYSTEM

## 6.1 Floating Objects

### Logs

* Move horizontally
* Safe to stand on

### Turtles

* Submerge periodically (critical mechanic)

---

## 6.2 Water Rules

* Player dies if not on object
* Player moves with object velocity

---

## 6.3 Submerging Turtles

### Behavior

```text
Visible → Safe
Blinking → Warning
Submerged → Deadly
```
