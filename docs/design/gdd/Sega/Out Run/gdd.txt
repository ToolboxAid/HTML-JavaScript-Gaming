# 3. 🚗 PLAYER SYSTEM

## 3.1 Controls

* Steering (left/right)
* Accelerator
* Brake
* Gear shift (Low / High)

---

## 3.2 Movement Model (ARCADE PHYSICS)

```text
velocity += acceleration (gear dependent)
velocity -= drag
velocity -= brakingForce
```

---

## 3.3 Gear System

### Low Gear

```text
highAcceleration
lowTopSpeed
```

---

### High Gear

```text
lowerAcceleration
higherTopSpeed
```

---

## 3.4 Steering Model

```text
turnRate = baseTurn * (1 - speedFactor)
```

* Higher speed reduces turning sharpness
* Requires braking for tight curves

---

## 3.5 Player States

| State    | Description     |
| -------- | --------------- |
| Driving  | Normal          |
| Off-road | Reduced control |
| Crashing | Spin-out        |
| Finished | End sequence    |
