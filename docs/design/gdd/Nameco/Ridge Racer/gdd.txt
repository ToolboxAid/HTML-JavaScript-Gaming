## 3.2 Movement Model (ARCADE DRIFT PHYSICS)

```text
velocity += acceleration
velocity -= drag
lateralSlip = steeringInput * driftFactor
```

---

## 3.3 Core Handling Properties

```text
Top Speed: High
Acceleration: Moderate
Grip: Low-mid (arcade tuned)
Drift Factor: High (signature mechanic)
```

---

## 3.4 Drift System (CRITICAL)

### Trigger Conditions

```text
IF speed > threshold
AND steering angle high:
    enter drift state
```

---

### Drift Behavior

```text
carAngle != velocityDirection
rear slides outward
```

---

### Exit Condition

```text
reduce steering
re-align velocity vector
```

---

* Smooth and controllable
* Long sustained slides
* Encourages flow through corners
