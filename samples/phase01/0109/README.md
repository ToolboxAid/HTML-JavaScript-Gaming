# Sample 0109 - Collision (AABB)

## Purpose
Demonstrates axis-aligned bounding box (AABB) collision detection with visual debugging.

## What it shows
- keyboard-controlled player movement
- static collision object
- overlap detection
- renderer-based debug visualization
- themed rendering consistency

## Controls
- Arrow keys: move player

## Behavior
The player box turns red when overlapping the static block.

## Notes
This is the foundation layer for future collision response systems.
