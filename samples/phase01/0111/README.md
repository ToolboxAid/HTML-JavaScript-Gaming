# Sample 0111 - Multiple Solids

## Purpose
Expands collision response to multiple blocking objects using a simple collision loop.

## What it shows
- one moving player
- multiple solid rectangles
- per-frame collision scanning
- active blocking solid highlight
- themed renderer debug output

## Controls
- Arrow keys: move player

## Behavior
When the player overlaps any solid, movement is reverted and the blocking solid is highlighted.
