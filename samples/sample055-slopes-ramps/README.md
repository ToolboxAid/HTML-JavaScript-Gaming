# Sample 055 - Slopes / Ramps

## Purpose
Demonstrates a first-pass ramp floor model where the actor snaps to a computed floor height on purple ramps.

## Controls
A/D or Left/Right to move. Space to jump.

## Notes
This build uses a simple math-based ramp floor and does not yet include generalized polygon collision.
