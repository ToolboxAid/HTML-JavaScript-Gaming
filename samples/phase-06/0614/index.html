Orbit

Arcade-clean orbit sandbox showing central-attractor trajectories with preset scenarios for stable, elliptical, inward-fall, and escape behavior.

Controls

1-4 select scenarios, P pauses or resumes, and R resets the current preset.

Preset Set

Stable Orbit, Elliptical Orbit, Fall Inward, and Escape Path are included so gravitational trajectory behavior can be compared under the same update loop.

Engine Classes Used

  • core / Engine
  • render / CanvasRenderer
  • input / InputService
  • scenes / Scene
  • theme / Theme
