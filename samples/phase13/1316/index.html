Network Sample A - Local Loopback / Fake Network

Sample-backed network diagnostics with no real transport. This validates read-only providers, passive overlay panels, and operator-safe command surfaces before host/client wiring.

Debug: ON

Debug overlay remains hidden by default. Use Shift+` for console or Ctrl+Shift+` for overlay.

Controls

Space/Enter sends a synthetic packet. C forces disconnect. R reconnects into handshake and sync.

Network Commands

network.help, network.status, network.latency, and network.trace [count].
