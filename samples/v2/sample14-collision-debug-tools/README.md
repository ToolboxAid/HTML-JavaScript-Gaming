# Sample14 - Collision Debug Tools

## Purpose
Adds visual debugging tools to help inspect which blockers are nearest and which one is actively overlapping.

## What it shows
- nearest candidate blocker highlighting
- active overlap highlighting
- live collision status text
- reusable debug-friendly obstacle layout
- themed rendering for diagnostics

## Controls
- Arrow keys: move player

## Behavior
Yellow shows the nearest blocker to the player. Red shows a blocker that is actively overlapping the player.
