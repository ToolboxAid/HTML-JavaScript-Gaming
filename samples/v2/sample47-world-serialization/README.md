# Sample47 - World Serialization

## Purpose
Introduces shared helpers for serializing and restoring world state.

## What it shows
- world snapshot serialization
- world snapshot restoration
- JSON-based persistence shape
- reusable save/load boundary
- world-state transfer format

## Controls
- Arrow keys: move actor
- KeyK: save snapshot
- KeyL: load snapshot

## Behavior
The scene can save and restore a JSON snapshot of the current world state.
