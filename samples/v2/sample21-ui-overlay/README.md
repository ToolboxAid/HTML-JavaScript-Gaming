# Sample21 - UI Overlay

## Purpose
Introduces a HUD and debug panel that render separately from world objects.

## What it shows
- screen-fixed HUD
- screen-fixed debug panel
- world rendering below UI
- ECS-based world entities
- clearer sample feedback

## Controls
- Arrow keys: move player

## Behavior
The player moves through the world while the HUD and debug panels remain fixed on the screen.
