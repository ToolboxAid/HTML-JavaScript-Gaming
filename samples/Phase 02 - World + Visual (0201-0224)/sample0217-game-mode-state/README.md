# Sample 0217 - Game Mode State

## Purpose
Introduces higher-level game mode flow for screens and runtime progression.

## What it shows
- title state
- play state
- pause state
- win state
- lose state

## Controls
- Enter: start/reset
- KeyP: pause toggle
- KeyW: win
- KeyL: lose

## Behavior
The sample shows how game flow state is managed above entity-level behavior.
