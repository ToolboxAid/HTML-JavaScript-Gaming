# Sample48 - Tile + Camera + Sprite Slice

## Purpose
Builds a tile-based playable slice using camera, action input, sprite-style rendering, and world snapshots.

## What it shows
- tilemap
- camera follow
- action input mapping
- sprite-atlas-style frame lookup
- world serialization
- integrated playable slice

## Controls
- Arrow keys or WASD: move actor
- KeyK: save snapshot
- KeyL: load snapshot

## Behavior
Move through the tile world, avoid walls, and reach the goal while using action input and snapshot support.
