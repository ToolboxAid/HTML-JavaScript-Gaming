Sample 1319 - Real Network + Live Dashboard

Minimal authoritative real-network sample. The client connects over WebSocket to a live server endpoint and renders authoritative player snapshots.

For this to work, you need to have the server running locally. You can start the server with
node samples/phase-13/1319/server/realNetworkServer.mjs
from the root of the repository.

Server health: http://127.0.0.1:4320/admin/network-sample-1319/health
Live dashboard: http://127.0.0.1:4320/admin/network-sample-1319/dashboard

Controls

Move with WASD or arrow keys. Client sends input to the authoritative server and renders real replicated snapshots.
