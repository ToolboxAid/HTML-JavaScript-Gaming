Sample 1317 - Host / Client Diagnostics

Sample-backed multi-role diagnostics that simulate host and client peers with read-only debug surfaces and no real transport layer.

Debug: ON

Debug overlay remains hidden by default. Use Shift+` for console or Ctrl+Shift+` for overlay.

Controls

Space/Enter inject host packet. 1 toggles Client A. 2 toggles Client B. H toggles Host.

Network Commands

network.help, network.connections, and network.replication.

  • tools/dev/plugins / createDebugPluginRegistry
