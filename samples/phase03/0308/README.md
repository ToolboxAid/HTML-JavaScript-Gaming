# Sample 056 - Gravity Zones

## Purpose
Demonstrates directional gravity fields using authored world rectangles.

## Controls
Arrow keys or WASD to add thrust and move between zones.
Press Space to jump opposite the current gravity direction.

## Notes
This sample intentionally keeps jump local to Sample 056.
It is a simple first pass so raised blocks in this sample are reachable.
