# Sample34 - Projectile Spawn System

## Purpose
Introduces reusable projectile spawning and update helpers.

## What it shows
- runtime spawning
- projectile motion
- projectile lifetime
- shared projectile helpers
- gameplay-oriented entity bursts

## Controls
- Arrow keys: move actor
- Space: fire projectile

## Behavior
The actor can spawn projectiles that move across the play area and expire automatically.
