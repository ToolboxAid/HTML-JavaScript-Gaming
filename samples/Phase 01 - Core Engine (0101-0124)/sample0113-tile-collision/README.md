# Sample 0113 - Tile Collision

## Purpose
Introduces tile-based collision using a solid grid rather than individual hand-placed rectangles.

## What it shows
- tile map as collision source
- solid and open tiles
- axis-separated movement response
- grid debug rendering
- transition toward map-driven gameplay

## Controls
- Arrow keys: move player

## Behavior
The player can move through open floor cells but is blocked by solid wall tiles.
