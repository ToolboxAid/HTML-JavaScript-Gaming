Frequently Asked Questions

What is Game Foundry Studio?

Game Foundry Studio is a browser-based place to plan, build, play, and share games with creator tools, game pages, docs, marketplace ideas, and account surfaces.

Which tool should I open first?

Start with Game Design Studio when you are shaping an idea, then move into Asset Studio, Animation Studio, Sound Studio, and the publishing tools as the project becomes more concrete.

Do I need an account to build?

The account pages show the intended creator profile, branding, controls, rating, and settings structure. Tool workflows can still be explored from the site nav while account services are wired in.

Where do assets and published games live?

Asset, marketplace, and publishing pages outline the destination structure for game media, reusable resources, and releases. The docs area covers manifest and publishing rules as they are formalized.

Need the build rules? Use the Docs page for manifests, tool guides, validation expectations, and publishing contracts.
