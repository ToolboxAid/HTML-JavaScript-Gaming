Theme V2 Overview

Purpose: Define the single approved styling surface for Game Foundry Studio LLC pages, Admin references, and first-class tool shells.

Approved Theme V2 ownership file: theme.css

Example usage: <link rel="stylesheet" href="../assets/theme-v2/css/theme.css">

Current consumers: Theme V2 Admin pages, root Tools pages, shared header and footer partials, and first-class tool surfaces when they consume the approved shell.

Status: Approved active surface.

Colors

Purpose: Own raw color values, semantic color variables, swatches, group accents, and color-family mappings.

Approved Theme V2 ownership file: colors.css

Example usage: <span class="brand-color-swatch swatch-orange"></span>

Current consumers: Branding references, Grouping Colors, Controls, tool group tiles, Admin navigation states, and status surfaces.

Status: Approved.

Typography

Purpose: Own type scale, heading behavior, body copy, code text, kicker text, and lede text.

Approved Theme V2 ownership file: typography.css

Current consumers: Page titles, section headings, navigation, footer groups, cards, forms, tables, and documentation pages.

Status: Approved.
