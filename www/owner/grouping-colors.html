Usage Notes

Rainbow Order

Toolbox Group view follows the Game Journey order from Idea #FF2D2D through Share #FF4F8B. Each group keeps a unique color.

Idea and Create

Idea #FF2D2D is for pre-project thinking and learning. Create #F59E0B follows creator workflow order: Game Hub, Game Crew, Game Configuration, then Tags.

Studio Team and Game Crew

Studio Team is the account-level roster. Game Crew is game-level assignment, and current Toolbox planning focuses on Game Crew inside Create.

Build Path Independence

Project Build Path keeps workflow order and project-specific readiness. Group colors are only for browsing and scanning tool categories.

Shared Theme V2 Classes

Owner pages should consume shared Theme V2 swatches and group classes from assets/theme-v2/css/colors.css. Pages should not define local group colors.
