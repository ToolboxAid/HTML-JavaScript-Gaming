Bouncing Ball

A minimal arcade-style ball motion test built as a standalone game: one ball, one rectangular arena, clean wall bounces, and no gravity.

Controls

Space or Enter starts. P pauses. R resets to the title state. Gamepads use south button to start and start to pause.

Gameplay Loop

Start the ball and watch it rebound around the bounded playfield. This build is about motion and collision stability only, with no gravity or scoring yet.
