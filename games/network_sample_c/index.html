Network Sample C - Divergence / Trace Validation

Deterministic mismatch and trace-validation showcase with timeline sequencing, divergence explanation notes, reproduction guidance, and checklist-driven verification.

Debug: ON

Debug overlay remains hidden by default. Open Play first, then use Debug Mode intentionally. Keyboard shortcuts: Shift+` for console and Ctrl+Shift+` for overlay.

Controls

Space/Enter add manual trace marker. D inject mismatch. R request reconcile. W refresh rewind prep. X execute rewind replay. V run validation.

Network Commands

network.help, network.divergence, network.trace [count], network.validate, network.sample.repro, and network.rewind.
