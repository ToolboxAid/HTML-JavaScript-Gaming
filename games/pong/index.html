Pong

Arcade-style Pong build with Tennis, Hockey, Handball, and Jai-Alai modes, paddle english, and keyboard/gamepad support.

Controls

W/S move the left paddle, and Hockey also uses A/D for left/right drift away from goal. Up/Down move the right paddle. Space or Enter serves and restarts. M changes mode. P pauses. X exits the current run back to the menu. Gamepads use the left stick or D-pad; south button serves, start pauses, and east/cancel returns to menu.

Modes

  • Tennis: classic head-to-head Pong.
  • Hockey: enclosed rink with goal slots at each end.
  • Handball: solo return practice against the far wall.
  • Jai-Alai: high-speed solo wall game with faster rebounds.
