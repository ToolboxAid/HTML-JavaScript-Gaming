Asteroids - Debug Showcase

Playable Asteroids-style arcade game and canonical debug showcase with player-select flow, high score, player scores, and ship-icon lives.

Debug: OFF

Enable debug with ?debug=1, then use Shift+` for console and Ctrl+Shift+` for overlay.

When debug is enabled, Asteroids auto-loads asteroidsshowcase.preset.default and keeps overlay hidden until you open it.

Keys

1 or 2 starts one-player or two-player. Left/Right rotate, Up thrusts, Space fires, and Enter returns to the menu after game over.

Debug Showcase

Debug surfaces are production-safe and opt-in. Launch with ?debug=1 (optional &debugMode=qa) to enable the Asteroids showcase panels and commands.

Use ?debugDemo=1 for one-click demo mode and &rememberDebug=1 to remember debug ON/OFF state for this browser.

Gameplay Loop

  • Clear each asteroid wave while surviving collisions and incoming fire.
  • Use thrust and rotation to line up shots, then split large rocks into smaller threats.
  • Manage UFO pressure, preserve lives, and push for the high score across one-player or alternating two-player runs.
