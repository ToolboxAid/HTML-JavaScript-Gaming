Projectile Lab

Arcade-clean projectile sandbox for spawning, travel, lifetime cleanup, repeated firing stability, and preset comparison.

Controls

1-5 select presets, Space or F fires, P pauses or resumes, and R resets the lab.

Preset Set

Single Shot, Fast Short-Life, Slow Long-Life, Burst, and Angled Shot are provided so projectile behavior can be compared under stable repeated update conditions.

Engine Classes Used

  • core / Engine
  • render / CanvasRenderer
  • input / InputService
  • scenes / Scene
  • theme / Theme
