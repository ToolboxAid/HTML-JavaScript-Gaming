# PR_26167_182-remove-provider-selection-runtime

## Summary

Removed active runtime provider selection from the local API/auth/product-data path. The runtime now uses one fixed server-owned path: Supabase Auth for auth/session and Supabase Postgres for product data.

Legacy provider selector environment variables are ignored and reported as safe diagnostics. Admin DB Viewer source/provider labels remain diagnostic-only and do not enable runtime switching.
