# Backlog Master

## Status Legend

- `[ ]` Planned
- `[?]` Wireframe
- `[.]` Building
- `[x]` Complete
- `[!]` Blocked
- `[-]` Deprecated

## Governance

Backlog item text is immutable after creation.

Allowed changes:
- add new items
- change status
- add notes
- add references

Prohibited without owner approval:
- delete items
- rename items
- rewrite item text
- reclassify items
