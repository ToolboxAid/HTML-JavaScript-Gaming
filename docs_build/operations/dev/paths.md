# Repo Path Contracts

## Active Paths
- ROADMAP_PATH = `docs_build/dev/roadmaps/`
- MASTER_ROADMAP = `docs_build/dev/roadmaps/MASTER_ROADMAP_ENGINE.md`
- PR_PATH = `docs_build/pr/`
- REPORTS_PATH = `docs_build/dev/reports/`

## Archive Paths
- DEV_OPS_ARCHIVE = `archive/v1-v2/docs_build/archive/dev-ops/`
- GENERATED_REPORTS_ARCHIVE = `archive/v1-v2/docs_build/archive/generated-reports/`
- PR_HISTORY_ARCHIVE = `archive/v1-v2/docs_build/archive/pr/legacy-pr-history/`

## Strict Rules
1. Active roadmap files must live in `docs_build/dev/roadmaps/`.
2. Do not keep duplicate active roadmap copies outside `docs_build/dev/roadmaps/`.
3. Transitional roadmap files outside `docs_build/dev/roadmaps/` must be archived.
4. Reports belong in `docs_build/dev/reports/`.
5. PR history belongs in `docs_build/pr/` or `archive/v1-v2/docs_build/archive/pr/legacy-pr-history/`.
