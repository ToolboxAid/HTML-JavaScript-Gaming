About Game Foundry Studio

Build the game. Ship the world. Bring the players.

Game Foundry Studio is a creator destination for the open web: creator tools, playable games, marketplace assets, tutorials, cloud saves and community discovery in one foundry-bright workspace.

Create

Forge sprites, sounds, scenes, input maps and prototypes with web-native tools built for fast iteration.

Play

Discover playable experiments, arcade releases and creator showcases without leaving the browser.

Share

Package assets, publish updates, learn from tutorials and grow a community around every finished build.

Open-web game studio

A creator playground with production instincts.

The direction is simple: make the browser feel like a living game foundry. Tools, games, tutorials, marketplace assets, saves and discovery should feel connected, energetic and ready for creators who want to move from idea to playable world without ceremony.
