# Game Foundry Studio Static Branding Site

Open `index.html` in a browser.

## Included
- `index.html` — modern landing page with hero, dual Play/Create entry, Trending Games, tools, and mascot section
- `brand.html` — brand kit, color palette, logo showcase, voice guidance
- `assets.html` — asset gallery for generated logo, mascot sheet, SVG marks, favicon, and game-card art
- `assets/css/styles.css` — responsive dark foundry theme
- `assets/images/game-foundry-studio-logo.png` — generated logo from this chat
- `assets/images/game-foundry-mascot-sheet.png` — generated mascot concept sheet from this chat
- `assets/images/*.svg` — existing logo mark, favicon, ForgeBot, and sample game art
- `assets/css/colors.css` — reusable color palette and color token source of truth

## Brand direction
- Name: Game Foundry Studio
- Tagline: Build. Play. Share.
- Palette: Iron Black, Foundry Space, Steel Gray, Molten Orange, Forge Gold, Electric Blue
- Mascots: ForgeBot, Spark, Pixel Smith, Foundry Bot
