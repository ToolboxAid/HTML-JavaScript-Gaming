CSS Ownership

colors.css
Approved Theme V2 color values, color-family variables, swatches, and group variables.
spacing.css
Theme V2 spacing, sizing, radius, shadow, z-index, icon, and layout tokens.
typography.css
Theme V2 text scale, font families, headings, body copy, and editorial text primitives.
layout.css
Theme V2 containers, page sections, navigation, footer, grid, split, and content layout primitives.
buttons.css
Theme V2 button and action-group primitives.
forms.css
Theme V2 form elements, field groups, field rows, and helper text primitives.
controls.css
Theme V2 control demos, switches, sliders, choice rows, and tool-group control accents.
panels.css
Theme V2 cards, surfaces, panel sections, media slots, callouts, and related content panels.
accordion.css
Theme V2 disclosure and accordion primitives.
status.css
Theme V2 feedback and event-log primitives.
tables.css
Theme V2 table baseline and data table primitives.
dialogs.css
Reserved Theme V2 dialog ownership file for future approved dialog primitives.

Design Rules

  • Use Theme V2 primitives before requesting new reusable styling.
  • Keep raw color values in colors.css.
  • Keep non-color design values in the appropriate Theme V2 ownership file.
  • Use current tool group names as the primary user-facing organization.
  • Use badge and tile colors as secondary visual cues.
