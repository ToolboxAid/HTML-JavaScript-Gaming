CSS Ownership

colors.css
Approved color values and color-family variables.
tokens.css
Spacing, sizing, typography, radius, shadows, z-index, icon sizes, and tool tile dimensions.
base.css
Reset, typography defaults, focus styles, accessibility helpers, and base element rules.
controls.css
Native HTML control scale, spacing, sizing, and typography.
gamefoundrystudio.css
Shared GameFoundry components and patterns.
pages.css
Page-specific layout helpers that consume shared tokens and colors.

Design Rules

  • Use shared CSS layers before adding page-specific styles.
  • Keep raw color values in colors.css.
  • Keep non-color design values in tokens.css.
  • Use current tool group names as the primary user-facing organization.
  • Use badge and tile colors as secondary visual cues.
