Getting Started

Screen and layout guidance for reviewing first-class Toolbox pages.

Desktop Target

Tools are built for 1440px and larger desktop workspaces.

Ideal Width

1920px is the ideal desktop width for wide tool reviews with small margins and a dominant center panel.

Minimum Comfortable Width

1440px is the minimum comfortable desktop width for the left panel, center panel, and right panel tool layout.

Panel Layout

The desktop layout is left margin, left panel, center panel, right panel, and right margin.

Smaller Screens

Smaller screens may use stacked or collapsed panels later.

Wireframe Status

This Learn page is guidance-only and does not add responsive runtime behavior.
