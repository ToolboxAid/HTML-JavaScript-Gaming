About Game Foundry Studio

Build, play and share open-web games without writing code.

Game Foundry Studio is a no-code, configuration-driven creator destination where game ideas move from GDD to manifest to engine. Creators shape content, rules, worlds and publishing plans through composition instead of custom user code.

Build

Create games through configuration, content and composition: scenes, sprites, rules, inputs, objectives and templates that the engine can understand.

Play

Test, preview, run, discover and enjoy games and examples in the browser while the same manifest-driven engine powers each experience.

Share

Publish games, assets, templates, tutorials, ratings and community knowledge so every useful pattern can help the next creator move faster.

GDD → Manifest → Engine

A config-driven engine for anyone with a game idea.

The long-term mission is to enable anyone to create games for the open web without writing code. Game Foundry Studio turns a design plan into structured manifests, then lets the engine run behavior from configuration instead of custom scripts.

The engine goal is zero user code: all gameplay behavior, content, presentation and publishing intent should be described through manifests, reusable templates and clear creator controls.
