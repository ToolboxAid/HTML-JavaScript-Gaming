# 1. 🧭 CORE GAME OVERVIEW

## 1.1 Game Pillars

* **Twin-Stick Precision Control**
* **Constant 360° Threat Pressure**
* **Civilian Rescue Risk/Reward**
* **Escalating Enemy Density**
* **Pure Reflex Survival**

---

## 1.2 Player Fantasy

You are the last human defender, protecting civilians from robotic enemies in a relentless assault.

---

# 2. 🎮 CORE GAMEPLAY LOOP

1. Spawn in arena
2. Move with left stick (or directional input)
3. Shoot independently with right stick
4. Destroy enemies
5. Rescue humans for bonus points
6. Survive wave
7. Next wave increases difficulty
8. Death → immediate respawn (if lives remain)

---
