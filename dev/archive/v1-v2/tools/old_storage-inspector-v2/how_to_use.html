Storage Inspector V2

Inspect and explicitly clear current-origin storage values without writing handoff data.

Use

Open the tool, choose a storage scope, filter by key or value text, select an entry to inspect its parsed JSON and raw value, delete one key, or clear session storage, local storage, tool state, or all storage when cleanup is needed.

Boundaries

The tool does not create repo handles, modify workspace manifests, or communicate state to another tool. Delete and clear actions affect only current-origin browser storage.
