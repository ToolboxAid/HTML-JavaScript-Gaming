# LEVEL_EDITOR_POLISH_ACCEPTANCE

## Acceptance Criteria
- both editors expose the same core load actions
- both editors expose Simulate and Exit Simulation
- Tile Map Editor can preview current map state
- Parallax Editor can preview current parallax motion/depth state
- sample loading works from each tool's local sample location
- no engine core API modifications
- reports are included for review

## Non-Goals
- full gameplay runtime inside editors
- merging both editors into one tool
- deep engine rendering rewrite
