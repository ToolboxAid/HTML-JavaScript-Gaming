# LEVEL_SHARED_PALETTE_AND_SWATCH_ACCEPTANCE

## Acceptance Criteria
- all tools visibly use the shared theme system
- SVG Background Editor exposes explicit Active Paint and Active Stroke selection flow
- SVG Background Editor has used-color swatches above the palette
- Sprite Editor has used-color swatches above the palette
- swatch clicks work like fast palette selection
- selected colors persist until intentionally changed
- no engine core API changes
