# Development Artifact Workspace

This folder owns generated non-report development artifacts.

- `tmp/` is the ignored location for local staging and temporary generated files.
- `zips/` is the ignored location for repo-structured ZIP artifacts.
- `logs/` is the ignored location for local generated logs.
- `generated/` is the ignored location for generated non-report artifacts.
- `test-results/` is the ignored location for Playwright and test result output.
- Report files belong in `dev/reports/` instead of this folder.
- Runtime product data must not be written here.
