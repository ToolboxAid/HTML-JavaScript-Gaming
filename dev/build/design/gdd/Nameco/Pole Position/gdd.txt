# 3. 🚗 PLAYER SYSTEM

## 3.1 Controls

* Steering (left/right)
* Accelerator
* Brake
* Gear shift (Low / High)

---

## 3.2 Movement Model (ARCADE PHYSICS)

```text id="pp1x9k"
velocity += acceleration (based on gear)
velocity -= friction
velocity -= brakeForce
```

---

## 3.3 Gear System (CRITICAL)

### Low Gear

```text id="pp2m4a"
highAcceleration
lowTopSpeed
```

---

### High Gear

```text id="pp3v7q"
lowerAcceleration
higherTopSpeed
```

---

## 3.4 Steering Model

```text id="pp4z2s"
turnRate = baseTurn * (1 - speedFactor)
```

* Higher speed → harder to steer
* Sharp curves require braking

---

## 3.5 Player States

| State    | Description     |

| Driving  | Normal          |
| Off-road | Reduced control |
| Crashing | Explosion       |
| Finished | Lap complete    |
