Here are **clean, copy/paste instructions** to install Playwright on another machine/page.

---

## 🧪 Install Playwright (VS Code / Windows / Node)

### 1. Open project folder

```powershell
cd C:\path\to\HTML-JavaScript-Gaming
```

---

### 2. Initialize Node (if not already)

```powershell
npm init -y
```

---

### 3. Install Playwright test runner

```powershell
npm install -D @playwright/test
```

---

### 4. Install browsers

```powershell
npx playwright install
```

---

### 5. Verify install

```powershell
npx playwright --version
```

---

## ▶️ Run your test

```powershell
npx playwright test dev/tests/ui/workspace-v2.asset-manager.spec.js
```

---

## 👀 Run with visible browser

```powershell
npx playwright test --headed
```

---

## 🐢 Slow motion (via config)

Create file at repo root:

```text
dev/config/playwright.config.cjs
```

```javascript
module.exports = {
  use: {
    headless: false,
    launchOptions: {
      slowMo: 500
    },
    trace: 'on'
  }
};
```

Then run:

```powershell
npx playwright test
```

---

## 🔍 View test replay

```powershell
npx playwright show-trace
```

---
